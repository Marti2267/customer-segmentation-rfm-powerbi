# customer-segmentation-rfm-powerbi
Customer Segmentation (RFM Analysis)

RFM-based customer segmentation project using Power BI to analyze customer value and retention.

This project focuses on customer segmentation using the RFM (Recency, Frequency, Monetary) model to analyze customer behavior and identify high-value segments.

Objectives

- Segment customers based on purchasing behavior

- Identify high-value and at-risk customers

- Support data-driven marketing and retention strategies

  Methodology

Customers were evaluated using three key metrics:

- Recency → time since last purchase

- Frequency → number of purchases

- Monetary → total revenue generated

Based on these metrics, customers were segmented into:

- Champions

- Loyal Customers

- Potential Loyalists

- At Risk

- Others

Tools & Technologies

- Power BI → data visualization and dashboarding

- DAX → KPI calculation and measures

- MySQL → data extraction and cleaning

Dashboard Overview

The dashboard includes:

- Customer distribution by segment

- Revenue contribution by segment

- Customer behavior analysis (Frequency vs Monetary)

- Top customers analysis and revenue concentration

Key Insights

- Champions generate over 60% of total revenue, highlighting a strong concentration of value

- Loyal Customers provide a stable revenue base

- At Risk customers show potential churn, suggesting targeted retention actions

- Top customers contribute significantly to total revenue, emphasizing the importance of customer loyalty

Business Impact

This analysis supports:

- Customer retention strategies

- Targeted marketing campaigns

- Customer value optimization
